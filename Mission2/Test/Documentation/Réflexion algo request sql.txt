Observations : 

dans le modèle actuel (6 possibilités d'affichage), si une combobox "ASX_4" possède un "tout", 
	on se retrouve dans l'affichage des blocs de droite "QQCH par Personne/Entretien/Stat".
ce qui conditionne le bloc final est la combobox AP_1.
ce qui conditionne le contenu de la première listbox sera l'AS possédant le "tout".
pour chaque champ de cette listbox on aura ensuite le choix d'un entretien dans la listbox
	suivante, qui fera varier les données de droite.

dans le cas des "Stats", la combobox AP_2 conditionne un type de requete particulier comme
	min, avg, max, count
